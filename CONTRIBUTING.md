# Contributing to S.U.A.S. Veteran Crisis QRF

Thank you for wanting to help. This repo powers the public site for **S.U.A.S. Veteran Crisis QRF** — a 501(c)(3) nonprofit working to combat veteran isolation. Every contribution, big or small, helps us reach more veterans.

## Ways to Contribute

You do not need to be a developer to help.

- **Designers**: logo, brand kit, page layouts, photo selection
- **Writers**: copy editing, story drafts, event recaps, donor appeals
- **Developers**: site code, Jekyll theme, accessibility fixes, custom domain setup
- **Photographers**: event photography we can use on the site
- **Veterans and supporters**: testimonials, story submissions, feedback

## Quick Start

1. **Browse open issues** — look for the `good first issue` label
2. **Comment on an issue** to claim it before starting work
3. **Fork the repo** and create a branch off `main`
4. **Make your change** and open a pull request
5. A maintainer will review and merge

If you find a problem but do not want to fix it yourself, **open an issue** describing what you saw.

## Content Guidelines

Because we serve veterans in crisis, please keep these in mind:

- **Veteran-first language**: respectful, direct, never patronizing
- **No clinical or detached tone**: we meet people where they are
- **No glorification of violence or service**: focus on connection and community
- **Accessibility**: write at a plain-language reading level
- **Crisis safety**: any page that could be read by someone in crisis should include the Veterans Crisis Line (988, press 1)

## Running the Site Locally

The site is built with Jekyll (GitHub Pages default).

```bash
# Install Ruby and Bundler, then:
gem install jekyll bundler
bundle exec jekyll serve
```

Open http://localhost:4000 to preview.

## Code of Conduct

Be respectful. We are a veteran-serving organization. We expect contributors to treat each other with the same care we extend to the veterans we serve.

Harassment, hate speech, or disrespect toward veterans, military families, or contributors will not be tolerated.

## Crisis Resources

If you or a veteran you love is in crisis right now:

- **Veterans Crisis Line**: Dial **988**, then press **1**
- **Text**: 838255
- **Chat**: [VeteransCrisisLine.net](https://www.veteranscrisisline.net/)

You are not alone.
